webpack学习

前端 webpack 2019/09/16

1.webpack基础配置

  1. webpack安装
  • 安装本地的webpack
  • npm install webpack webpack-cli -D
  1. webpack可以进行0配置
  • 打包工具 -> 输出后的结果（js模块）
  • 打包（支持我们的js模块化）npx webpack
  1. 手动配置webpack
  • 默认配置文件的名字 webpack.config.js

2.webpack打包出的文件解析

  • npx webpack –config webpack.config.my.js
  • npm run build – –config webpack.config.my.js（– 传配置）

3.Html插件

  • npm i webpack-dev-server -D（下载webpack-dev-server）开发服务器的配置
  • npx webpack-dev-server（运行webpack-dev-server）
  • “dev”:”webpack-dev-server”
  • npm i html-webpack-plugin -D

4.样式处理

  • npm i css-loader style-loader -D

  • npm i less less-loader -D

  • CSS抽离

  • npm i mini-css-extract-plugin -D（为每个引入 CSS 的 JS 文件创建一个 CSS 文件）

  • 自动加上“-webkit-”之类的css前缀，我们需要用到postcss-loader和它的插件autoprefixer
  • npm i postcss-loader autoprefixer -D
  • 在根目录下新建postcss.config.js
  • 压缩css代码
  • npm install optimize-css-assets-webpack-plugin -D

总结：webpack打包优化需要不断的去理解插件的作用以及造成的影响，不要滥用插件。

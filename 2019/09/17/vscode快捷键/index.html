vscode快捷键

工具 vscode 2019/09/17

1.代码折叠

  1. （折叠最内层的、可以被折叠的代码块）Ctrl + Shift + 左方括号
  2. （展开最内层的、可以被展开的代码块）Ctrl + Shift + 右方括号
  3. （所有可以被折叠的代码一次性全部折叠起来）Ctrl + K，Ctrl + 0
  4. （所有可以被展开的代码一次性全部展开起来）Ctrl + K，Ctrl + J
  5. （基于语言定义的代码折叠）// region Main // endregion

2.单文件搜索和替换

  1. （快速地调出搜索窗口）Ctrl + F
  2. （在搜索结果之间自上而下跳转）回车键
  3. （在搜索结果之间自下而上跳转）shift + 回车键
  4. （调出了搜索框,但光标依然是在编辑器当中，而不是在搜索框中）F3
  5. （在搜索结果之间自上而下地循环跳转）F3
  6. （实现自下而上的跳转）Shift + F3
  7. （搜索框最右侧的三个配置按钮）
  8. （第一个是大小写敏感Case） Alt + C
  9. （第二个是全单词匹配Word） Alt + W
  10. （第三个是正则表达式匹配Regular Expression） Alt + R
  11. （调出替换窗口）Ctrl + H
  12. （调出替换窗口）Ctrl + Shift + 1
  13. （调出替换窗口）Ctrl + Alt + 回车键
  14. （单文件搜索配置editor.find.autoFindInSelection）选中了多行文本，开始搜索时，自动在这几行代码里进行搜索

3.多文件搜索和替换

  1. （快速地调出多文件搜索窗口）Ctrl + Shift + F
  2. （多文件搜索配置search.collapseResults）alwaysExpand直接展开结果

4.编辑器设置

  1. （控制整个行号显示）editor.lineNumbers
  2. （让编辑器将所有的空格符、制表符等全部都渲染出来）editor.renderWhitespace: all
  3. （缩进参考线）editor.renderIndentGuides
  4. （垂直标尺）”editor.rulers”: [120]
  5. （小地图 Minimap）editor.minimap.enabled
  6. （光标的样式）”editor.cursorBlinking” “editor.cursorStyle” 和 “editor.cursorWidth”
  7. （当前代码行的行号下的背景色）editor.renderLineHighlight: “all”
  8. （创建一个新文件的格式）files.defaultLanguage: “markdown”
  9. editor cursor， 是跟光标渲染和多光标相关的设置；
  10. editor find， 是与编辑器内搜索相关的设置；
  11. editor font， 是与字体有关的设置；
  12. editor format， 是代码格式化；
  13. editor suggest， 是和自动补全、建议窗口等相关的配置。

5.文件管理

  1. （调出最近打开的文件夹的列表）Ctrl + R，Ctrl + 回车
  2. （将文件夹添加到工作区）add folder to workspace
  3. （将工作区另存为）save workspace as
  4. （切换窗口）Switch Window

6.命令面板

  1. （打开命令面板快速访问符号）Ctrl + Shift + O
  2. （打开命令面板）Ctrl + Shift + P
  3. （文件资源管理器）Ctrl + Shift + E
  4. （跨文件搜索）Ctrl + Shift + F
  5. （源代码管理）Ctrl + Shift + G
  6. （启动和调试）Ctrl + Shift + D
  7. （管理扩展）Ctrl + Shift + X
  8. （查看错误和警告）Ctrl + Shift + M
  9. （打开侧边栏）Ctrl + B

7.插件

  1. （可以可视化看分支）GitLens
